# Marketplace Screenshots

These images are referenced by the **Screenshots** section of the main `README.md`
and shown on the VS Code Marketplace listing.

| File | Shows |
|------|-------|
| `diagnostics.png` | An unprotected `OpenAI()` call with the inline PromptGuard diagnostic squiggle and the **PromptGuard: Apply Transformations** quick-fix code action open. |
| `status-bar.png` | The bottom status-bar indicator in its three states: **Active**, **Disabled**, and **Not initialized**. |
| `sidebar.png` | The PromptGuard Activity Bar view expanded: Status, Managed Files, and the Actions group (Scan / Disable / Set API Key / Select Project). |

The current images are **faithful renders** built from the extension's actual UI
strings, the VS Code Dark+ palette, and the official `@vscode/codicons` icon set
(2560×1440, 2× DPI). To replace any of them with a live screen capture, just keep
the exact filenames — the Markdown references in `README.md` pick them up
automatically.
